This project analyzes restaurant data using machine learning and data visualization techniques.
It covers four major tasks: rating prediction, recommendation system, cuisine classification, and location-based analysis.
The dataset is preprocessed with proper handling of missing values and categorical encoding.
Regression models are used to predict restaurant ratings with strong accuracy.
A content-based recommendation system suggests restaurants based on cuisine, price range, and city.
Cuisine classification is performed using Logistic Regression and Random Forest with high F1-scores.
